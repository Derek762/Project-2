Cuphead-Style Boss Fight

How to Play

  • Move: Left/Right Arrow Keys
  • Jump: Up Arrow Key (jump high enough to reach platforms)
  • Glide: Hold Up Arrow while falling to slow your descent (limited time)
  • Shoot: Z Key (max 3 bullets at once)
  • Drop Through Platform: Down Arrow Key (while standing on a platform)
  • Restart: R Key (after win/lose)

Defeat the boss by dodging attacks and shooting!
Note: You must run this file on a local server (not by double-clicking).
To play: Use a tool like Live Server in VS Code, or run python3 -m http.server in this folder and open http://localhost:8000 in your browser.
